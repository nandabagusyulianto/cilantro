# Contributing to Cilantro

Thank you for your interest in contributing to Cilantro.

## Getting Started

1. Fork the repository
2. Clone your fork
3. Install dependencies: `bun install`
4. Create a new branch: `git checkout -b your-feature`

## Development

```bash
bun run dev
```

## Before Submitting

Format your code:

```bash
bun run format
```

## Commits

Use [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/):

```
feat: add multiple account support
fix: resolve steam guard prompt issue
docs: update readme
```

## Pull Requests

- Keep changes focused and minimal
- Describe what you changed and why
